# Account Policies

## 📚 Aula 2 — Políticas de Contas

As **políticas de contas** definem regras que controlam como contas de usuários são criadas, utilizadas e protegidas dentro de um sistema.

Essas políticas ajudam a garantir:

- segurança
- controle de acesso
- rastreabilidade de atividades

---

# Objetivos da Aula

- entender funcionamento das contas
- compreender atributos de contas
- conhecer políticas de segurança aplicadas a usuários
- entender controles de acesso baseados em identidade

---

# Atributos de Contas

Os **atributos de conta** são informações associadas a uma identidade digital.

Exemplos:

- nome do usuário
- email
- departamento
- nível de acesso
- histórico de login

Esses atributos ajudam na:

- auditoria
- gerenciamento de permissões
- identificação de usuários

---

# Controle de Contas

Organizações precisam controlar todo o ciclo de vida das contas.

Isso inclui:

1. criação da conta
2. atribuição de permissões
3. monitoramento de atividades
4. revogação de acesso

---

# Segurança de Contas

Boas práticas incluem:

- uso de autenticação forte
- revisão periódica de permissões
- auditoria de acessos
- remoção de contas inativas

---

# Auditoria de Contas

Auditorias ajudam a identificar:

- acessos indevidos
- contas não utilizadas
- privilégios excessivos

Auditorias periódicas aumentam o **controle e a transparência da segurança**.

---

# Gestão de Ciclo de Vida da Conta

O ciclo de vida de uma conta inclui três fases principais:

## 1. Criação

A conta é criada com permissões iniciais.

---

## 2. Uso

Durante o uso da conta podem ocorrer:

- mudanças de permissões
- redefinição de senhas
- auditorias de acesso

---

## 3. Encerramento

Quando o usuário deixa a organização:

- a conta deve ser desativada
- credenciais devem ser revogadas
- acessos devem ser removidos

---

# Importância das Políticas de Conta

Sem políticas adequadas, uma organização pode enfrentar riscos como:

- acesso não autorizado
- abuso de privilégios
- vazamento de dados
- ataques internos

Por isso, políticas de contas são fundamentais para uma **estratégia de segurança eficaz**.
